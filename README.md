#  Project Overview
This project is an AI driven vision prototype developed as a high school computer science extracurricular project. It aims to tackle indoor air pollution and respiratory health risks (such as Legionnaires' disease) caused by neglected, dirty air conditioning systems. 
The idea of this project came from my own concern about lung diseases in general, and indoor air pollution in particular. 
The application utilizes Deep Learning to detect mold/dust contamination and intelligently forecast the remaining lifespan of AC filters.

#  Project Timeline
  Week 1 (MVP v1.0): Data collection and training a Convolutional Neural Network (CNN) to classify clean vs. moldy AC units.
  Week 2 (Upgrade v2.0): Collecting time-series dust images and building a Regression Model to predict filter lifespan.
  Week 3 (Deployment): Building an interactive web interface using Streamlit and deploying the live application online.
